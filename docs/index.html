The Problem

UI

UI Tools

Test the CRM, but can't place real calls

Make calls, but can't verify what the agent sees

API Tools

Query records, but can't coordinate timing

QA

Manual QA

Expensive, slow, and non-reproducible

What Audrique Does

One framework coordinates all three layers — because contact center bugs live at the boundaries.

1

Visual Scenario Builder

Build voice tests in a 7-step wizard. Configure IVR routing, supervisor monitoring, and assertions — no code required.

2

Real Voice Calls

Places actual outbound calls through Amazon Connect + Twilio. Navigates IVR menus with speech detection and DTMF.

3

CRM Verification

Verifies the Salesforce agent sees the right screen pop, call type, and VoiceCall record — all via Playwright.

4

Supervisor Monitoring

Opens Command Center, monitors Queue Backlog and In-Progress Work tabs. Detects call routing in real time.

5

Three Video Streams

Records agent, CCP, and supervisor screens simultaneously. Generates timeline-synced video evidence.

6

Session Management

Federation API auth for Connect CCP — no SAML login. Auto-refresh expired sessions before each run.

How It Works

Three parallel streams coordinated by a single test scenario.

SF
Agent Stream

Salesforce Lightning — Playwright

Login to Omni-Channel, sync telephony provider, wait for incoming call offer, accept, verify VoiceCall screen pop.

CCP
CCP Stream

Amazon Connect CCP — Playwright

Load CCP with federation token cookies. Place outbound call via Twilio. Navigate IVR with speech detection and DTMF digits.

SUP
Supervisor Stream

Command Center — Playwright

Open Queue Backlog + In-Progress Work tabs. Poll both in parallel. Detect call routing to target queue.

REC
Evidence

Three Parallel Video Recordings

All streams record simultaneously. Timeline timestamps synchronize the recordings into a single evidence video.

Quick Start

Up and running in under 5 minutes.

Terminal
# Clone the repo
git clone https://github.com/snehalsurti12/audrique.git
cd audrique

# Install dependencies
npm install
npx playwright install chromium

# Start Scenario Studio (visual builder)
npm run studio

# Open http://localhost:4200 and build your first test
