Why Audrique?

Existing tools test one layer at a time. A call connects, but nobody checks if the CRM updated. A UI test passes, but nobody verified the call actually rang. Audrique tests all three layers simultaneously.

Without Audrique
  • UI tools test the CRM, but can't place real calls
  • Telephony scripts make calls, but can't verify what agents see
  • API tools query records, but can't coordinate timing
  • Manual QA is expensive, slow, and non-reproducible
With Audrique
  • One test places a real call and verifies the CRM simultaneously
  • Agent screen pops, call routing, and records validated in a single run
  • AI agent conversations tested with automated persona simulation
  • Reproducible, automated, with video evidence of every run

Capabilities

One framework that coordinates telephony, CRM, and backend — because contact center bugs live at the boundaries.

Real Call Automation

Place and receive real voice calls through Amazon Connect and Twilio. Navigate IVR menus with speech detection and DTMF input.

CRM Verification

Verify that Salesforce agents see the correct screen pop, call type, and VoiceCall record — all in real time via Playwright.

AI Agent Validation

Test Agentforce and other AI agents with LLM-driven persona simulation. Validate topic routing, actions, and resolution flows.

No-Code Scenario Builder

Build voice test scenarios in a visual 7-step wizard. Configure call routing, supervisor monitoring, and assertions without writing code.

Video Evidence

Record agent, CCP, and supervisor screens simultaneously. Generate timeline-synced video evidence for every test run.

Results Dashboard

Track test run history, failure diagnostics, and pass-rate trends in a built-in web dashboard with PostgreSQL-backed storage.

How It Works

Three parallel streams coordinated by a single test scenario.

Agent Stream

Salesforce Lightning

Log in to Omni-Channel, sync telephony, accept incoming call, verify VoiceCall screen pop and record fields.

Caller Stream

Amazon Connect + Twilio

Place outbound call via Twilio, navigate IVR with DTMF or speech, or simulate AI-driven customer conversation.

Supervisor Stream

Command Center

Monitor Queue Backlog and In-Progress Work tabs in real time. Detect call routing to the target queue and agent.

Evidence

Synchronized Recordings

All streams record simultaneously. Timeline-synced video evidence, assertions, and full conversation transcripts.

Quick Start

Up and running in under 5 minutes.

Terminal 
# Clone the repo
git clone https://github.com/snehalsurti12/audrique.git
cd audrique

# Install dependencies
npm install
npx playwright install chromium

# Start Scenario Studio (visual builder)
npm run studio

# Open http://localhost:4200 and build your first test
